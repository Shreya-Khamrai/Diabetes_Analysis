# Diabetes_Analysis
Built a diabetes prediction model using the Pima Indians Diabetes dataset (768 rows, 9 columns). I used KNN with features including Glucose, BMI, Blood Pressure, Insulin, Age, and Skin Thickness. I handled invalid 0 values, treated NaN values, standardized the data, split it into 70% training and 30% testing sets, and achieved 75.76% accuracy.
